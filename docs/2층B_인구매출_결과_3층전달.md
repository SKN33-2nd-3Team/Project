# 2층-B 결과 요약 — 인구 → 매출

작성: 2026-07-17 | 관련 Issue: #6
재현: `py scripts/layer2_b_population_sales.py`
분석 구간: 2023Q1~2025Q3 (분기 기준)

## 분석 목적과 방법

유동·상주·직장인구의 **증감률**과 매출 **증감률** 사이의 단독 연관을 각각 확인했다.

- 증감률 산식: `(현재값 - 직전 2분기 평균) ÷ 직전 2분기 평균 × 100`
- 매출 증감률: `상권×업종` 셀 단위
- 인구 증감률: `상권×분기` 단위에서 계산 후 같은 상권·분기의 매출 셀에 병합
- 분석 전 공통 제외 규칙 적용: 법무·회계·부동산·전문서비스 업종그룹, 핸드폰 세부업종
- 매출 기록이 있는 셀만 사용
- 2021~2022년은 검정 표본에서 제외했고, 2025Q4는 테스트 타깃으로 보존했다.

## 계절성·민감도 확인

- 유동인구는 2분기에서 `r=0.015`, `p<0.001`이지만 효과 크기가 매우 작고 다른 분기에서 재현되지 않아 단독 유효 신호로 보지 않았다.
- 상주·직장인구는 데이터 수집 주기 특성상 일부 분기에서 증감률이 모든 상권에 동일해 변동이 0이었다. 따라서 상주인구 2·3분기, 직장인구 2·3분기의 같은 분기 Pearson 상관은 계산 불가다.
- 코로나 기준값 영향을 줄이기 위해 2023Q1·Q2를 제외한 민감도 분석에서도 유동 `r=0.001`, 상주 `r=-0.000`, 직장 `r=0.004`로 결론이 변하지 않았다.

2층의 무효 판정은 제거를 의미하지 않는다. 설계서 §3에 따라 3층 조합 회귀에는 세 인구 변수를 모두 포함해 재검증한다.

## 해석·한계

1. 이 결과는 인구 변화가 매출 변화에 **영향을 준다**는 인과 해석이 아니라, 단독 선행 신호가 약했다는 관측 결과다.
2. 인구 값은 같은 상권·분기의 업종 행마다 반복된다. 따라서 p-value는 독립 관측을 가정한 값보다 작아질 수 있어, 판정은 p-value보다 Pearson r의 크기·방향을 중심으로 했다.
3. `sales_monthly_sales`는 카드 기반의 분기별 월 평균 명목 매출액이며 물가를 보정하지 않았다.
4. 상주·직장인구의 일부 분기 증감률에 변동이 없는 데이터 구조는 같은 분기 비교의 제약으로 발표 시 명시한다.
