# 1층·2층-A2 결과 요약 — 3층(조합 확인) 전달용

작성: 2026-07-17 | 담당 파트: 1층 현황(EDA) + 2층-A2 업종→매출
재현: `py scripts/layer1_status.py`, `py scripts/layer2_a2_industry_sales.py`

## 요소별 판정 (설계서 §3 3층의 최종 판정표 입력)

| 요소 | 판정 | 근거 |
|---|---|---|
| **업종그룹 → 매출 증감률** | **① 단독 유의 (계절 통제 조건부)** | Kruskal-Wallis. 전체 기간 한꺼번에 ε²=0.002(미미)이나, **같은 분기끼리 비교 시 ε²=0.012~0.020**(1~3분기). 4분기만 ε²=0.003으로 약함 |

- 함의: 4층 모델에 **업종그룹 더미 + 분기 더미를 반드시 함께** 투입. 분기 더미 없이는 업종 효과가 계절 효과에 묻힘
- 3층 요청사항: **업종그룹 × 분기 교호작용 확인 필요.** 그룹 간 차이의 방향이 분기마다 뒤집힘 (1분기: 소매 급락·교육 유일 플러스 / 4분기: 교육이 최저 -4.5%)

## 1층 현황 핵심 수치 (분기 기준, 가중 집계, 분모=점포수)

- 연도별: 2022 최저(2.6%대) → 2023~24 급등(3.0~3.2%대) → 2025 소폭 하락. **모든 상권유형·구에서 동일 패턴** → 코로나 구간(2021~22) 학습 제외 결정의 근거
- 상권유형: 골목상권이 전 기간 1위(2024년 3.63%), 관광특구 최저. 5년 내내 순위 불변 → 상권유형은 폐업률과 안정적 연관 신호 (2층-A1 참고)
- 업종그룹: 외식·주점 압도적 1위(분기 4.3~5.2%), 타 그룹의 약 2배
- 구별: 관악·강서·강북 상위 / 중구·종로·용산 하위. 동별 상·하위는 `reports/tables/` CSV 참조 (극단값 동은 점포 수 확인 후 인용할 것)
- 계절성: 1분기 높고 4분기 낮은 패턴 반복 → 폐업률에도 분기 더미 필요

## 2층-A2 부가 발견

- 2023~25 매출 증감률 중앙값이 **전 업종그룹에서 0 이하** (명목 기준 — 물가 미보정이라 실질은 더 하락)
- 하락폭 최대: 가정·자동차·기타소매, 패션·개인·문화소매 (각 -3.1%)
- 가정·자동차·기타소매의 1분기 급락(-12.2%)은 이상치가 아니라 **계절 구조** — 자전거(-27%→2분기 +66%), 인테리어·철물점(겨울 시공 비수기), 가전(연말 특수 직후) 주도. §2.3 "같은 분기끼리 비교" 규칙의 실증 사례

## 데이터 주의사항 (3·4층 공통)

1. **제외 규칙 필수 적용**: 모든 스크립트는 로드 직후 `src/data_rules.py`의 `apply_exclusions()` 호출 (법무·회계·부동산·전문서비스 그룹 + 핸드폰 업종, 30,486행). usecols에 `industry_group`·`industry_name` 필수
2. **매출 기록은 전체 행의 약 30%** (sales_record_present=true인 40만 행). 매출 피처는 이 부분집합 기준
3. `sales_monthly_sales` = **분기별 월 평균 매출액** (카드 기반 추정, 명목 금액). 금액 인용 시 단위 명시
4. 증감률은 `src/growth_rate.py` 공용 함수 사용 (분기 연속성 검증 포함). 셀 단위 매출 증감률은 `data/interim/sales_growth.csv` 재사용 가능
5. 폐업률 집계는 `src/closure_rate.py` 사용 (분모=stores_store_count, 팀 결정 ②). 데이터의 기존 폐업률·개업률 컬럼은 분모가 달라(유사점포수) 사용 금지
6. 업종 분석·더미는 **업종그룹(7개) 단위** (팀 결정 ④). 세부업종은 행 식별용만
