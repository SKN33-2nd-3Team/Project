# 서울시 민생경제의 버팀목, 소상공인을 잡아라

서울시 상권분석서비스 데이터(2021~2025년, 분기 단위)를 활용하여 상권·업종·매출·인구
요소와 폐업률의 연관 구조를 분석하고, **상권×업종그룹 단위의 다음 분기 폐업률을 회귀
모델로 예측**하는 프로젝트입니다.

서비스 사용자는 서울시 및 자치구의 소상공인 정책 담당자입니다. 예측 결과는 폐업 위험이
상대적으로 높은 상권·업종 조합을 우선 점검하고 지원 대상을 선정하기 위한 참고 자료로
사용합니다.

이 프로젝트에서 `폐업 = 이탈`은 고객 이탈 예측 방법론을 공공정책 도메인에 적용하기 위한
비유입니다. 본 모델은 개별 점포의 폐업 여부나 폐업 확률을 예측하지 않으며, 집계된
상권·업종 단위의 다음 분기 폐업률 지표를 예측합니다. 제출일은 2026-07-22입니다.

## 예측 문제 정의

- 사용자: 서울시 및 자치구 소상공인 정책 담당자
- 예측 단위: 상권×업종그룹×분기
- 예측 시점: t분기 데이터가 확정된 시점
- 입력 피처: t분기까지 확인 가능한 상권·업종·매출·인구 특성
- 타깃: t+1분기의 가중 폐업률
- 문제 유형: 회귀
- 주요 평가 지표: MAE
- 활용 방식: 예상 폐업률과 위험 순위를 이용한 정책 점검 대상 선정

고위험 여부는 모델이 직접 학습하는 0/1 타깃이 아닙니다. 회귀 모델이 예측한 다음 분기
폐업률을 기준으로 상위 위험 구간을 정책적 후처리를 통해 구분합니다.

## 실행

```bash
pip install -r requirements.txt
py scripts/layer1_status.py            # 1층: 현황 차트·구/동 현황표
py scripts/layer2_a2_industry_sales.py # 2층-A2: 업종그룹→매출 검정
```

- 공용 함수: `src/closure_rate.py`(폐업률 가중 집계), `src/growth_rate.py`(증감률 산식),
  `src/data_rules.py`(팀 제외 규칙 — 모든 스크립트가 로드 직후 호출)
